        Иногда возникает ситуация, когда доступ в интернет ограничен, например, заблокированы все порты кроме http или запрещён UDP трафик. На этот случай у нас есть возможность подключения к VPN по 443 (TCP) и 8090 (альтернативный HTTP) порту.

        Для подключения по TCP достаточно выбрать соответствующий тип конфигов во время подключения, а для HTTP отредактировать любой желаемый конфигурационный файл TCP,заменив в нем 443 порт на 8090.

        NEW Выбор протокола подключения доступен в нашей собственной программе для работы с VPN. В данный момент доступна версия для Windows (в будущем и для других систем).
